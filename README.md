Advanced Time Series Forecasting with Attention

This project implements a multivariate time series forecasting model using a Seq2Seq LSTM with an Attention mechanism.

-------------------------------------------------------------------

Key Features

Synthetic multivariate time series generation

Data preprocessing and scaling

Rolling-origin cross-validation

Seq2Seq LSTM with custom Attention layer

Baseline LSTM comparison

Evaluation using RMSE, MAE, and MAPE

---------------------------------------------------------------------
Tech Stack:

Python, NumPy, Pandas, Scikit-learn, TensorFlow/Keras
